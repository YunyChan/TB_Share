# 淘宝分享(TB_Share) #

淘宝商品链接去跟踪一键复制工具

## 特性 ##

* 在浏览器商品详情页时，点击扩展图标将自动过滤链接并添加到系统粘贴板中
* 生成商品链接二维码

## 安装 ##

+ [Chrome Web Store](https://chrome.google.com/webstore/detail/khpadjfpaalkmcgchahacadodhkiadle)
